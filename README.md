# visual-dsa-lab

Dark, AlgoMaster-style curriculum lab for interview DSA walkthroughs — animated step-by-step visuals for arrays, linked lists, and trees.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Engines

| Engine | Algorithms |
|--------|------------|
| Arrays | Bubble Sort, Two Pointers (pair sum), Kadane (Max Subarray) |
| Linked List | Reverse List, Insert at Position |
| Trees | BST Insert, Inorder Traversal |
| Graphs | Valid Path (BFS) |

Each lab: change **input only** → **Run** → scrub steps. Monaco shows **read-only Python** (no edit, no execute) with line highlights synced to the current step.

## Stack

Next.js, TypeScript, Tailwind, Zustand, GSAP (arrays), Motion + React Flow (lists/trees/graphs), D3 (tree layout), Monaco Editor.
